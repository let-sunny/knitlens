# Pull Request Rules

Pull requests generated by agents must include the following sections.

## Summary

Short explanation of the change.

## Context

Why this change is needed.

## Changes

List of the main changes.

Example

- add project creation screen
- implement pattern analysis API
- add Supabase project model

## Related Docs

Reference relevant project documentation when applicable.

Examples

docs/product/one-pager.md
docs/architecture/screen-flow.md
docs/architecture/data-model.md

## Follow-ups

Future improvements or work planned next.
